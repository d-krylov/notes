## LLDB

Reading registers:
```
register read
```
Run with arguments
```
lldb -- ExeFile -arg1 -arg2
```
Step over
```
n
```
Step into
```
s
```

## GDB

Run with arguments
```
gdb --args ExeFile -arg1 -arg2
```

Show local variables in current frame
```
info locals
```
